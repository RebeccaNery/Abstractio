# Ideias — Abstractio

## Estrutura de cada Missão

Cada missão seguirá sempre o mesmo esquema:

1. **Teoria** — explicação didática do conceito, com exemplos em Python
2. **Interação Visual / Jogo** — componente interativo que reforça o conceito de forma lúdica
3. **Exercício Prático** — questão de múltipla escolha para consolidar o aprendizado
